# Sales_Dashboard-Using-Power-Bi
<img src="Screenshot 2026-06-08 131425.png">

Analyzed E-commerce sales data using power Bi by creating an interactive dashboard.
Used different types of visulization (bar chart, donut chart, slicers, card ).

Key insights included :

Sales data by Quarter,
Total order analysis,
Total Quantity sold,
Profit Tracking & Monthly Profit Trends,
Profit by subcategory,
Total Sales by state.

Tools & Skills used:

Power Bi, Power Query , Data Modeling , Data visualization, Dashboard design.
